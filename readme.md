# Roots & Ruins

Roots & Ruins is a farming and adventure game built with Godot Engine 4.x. The project features farming, exploration, day/night cycles, inventory management, and a dialogue system. It is designed for extensibility and learning, with modular scripts and assets.

## Features

- **Farming & Collecting:** Plant crops, gather resources, and manage your farm.
- **Adventure & Exploration:** Explore levels, interact with NPCs, and discover secrets.
- **Day/Night Cycle:** Dynamic time system affecting gameplay.
- **Inventory System:** Simple inventory management for items and resources.
- **Dialogue System:** Interactive conversations with NPCs, supporting localization.
- **Modular Components:** Easily extendable scripts and resources.

## Folder Structure

- `scenes/` — Main scenes, levels, characters, components, UI.
- `scripts/` — Game logic, managers, state machines.
- `dialogue/` — Dialogue balloons, themes, conversations.
- `addons/dialogue_manager/` — Dialogue Manager plugin for Godot.
- `assets/` — Sprites, tiles, audio, UI, fonts.
- `audio/` — Music and sound effects.
- `tilesets/` — TileSet resources.
- `resources/` — Data resources for nodes, scenes, tilemaps, and saves.

## Getting Started

### Prerequisites

- [Godot Engine 4.x](https://godotengine.org/download)
- Git

### Clone the Repository

```sh
git clone https://github.com/Sases-Chaudhary/Roots-Ruins.git
cd Roots-Ruins
```

### Pull Latest Changes

```sh
git pull
```

### Open in Godot

1. Launch Godot Engine.
2. Select the project folder (`Roots-Ruins`).
3. Open the project.
4. Run `scenes/main_scene.tscn`.
